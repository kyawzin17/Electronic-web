## Hall Effect Sensor (သံလိုက်အာရုံခံကိရိယာ)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Hall Effect Sensor ဆိုသည်မှာ သံလိုက်စက်ကွင်း (Magnetic Field) ကို တုံ့ပြန်ပြီး လျှပ်စစ်ဗို့အား ပြောင်းလဲထုတ်ပေးသည့် ကိရိယာဖြစ်ပါသည်။ ၎င်းသည် Edwin Hall ရှာဖွေတွေ့ရှိခဲ့သော Hall Effect နည်းစဉ်အပေါ် အခြေခံထားသည်။ ဤ Sensor များကို အရာဝတ္ထုများ၏ အနီးကပ်တည်ရှိမှု၊ အမြန်နှုန်းနှင့် လျှပ်စီးကြောင်း (Current) တိုင်းတာခြင်းတို့တွင် အဓိကအသုံးပြုသည်။ ၎င်းတို့သည် ရုပ်ပိုင်းဆိုင်ရာ ထိတွေ့မှုမလိုဘဲ အလုပ်လုပ်နိုင်သဖြင့် ဖုန်ထူသောနေရာ သို့မဟုတ် ရေစိုသောနေရာများတွင် Optical sensor များထက် ပိုမိုခိုင်ခံ့စိတ်ချရသည်။

---

### သမိုင်းကြောင်း (History)
Hall Effect ကို ၁၈၇၉ ခုနှစ်တွင် **Edwin Hall** က စတင်တွေ့ရှိခဲ့သည်။ ဆယ်စုနှစ်ပေါင်းများစွာ ဓာတ်ခွဲခန်းထဲတွင်သာ ရှိနေခဲ့သော်လည်း ၁၉၅၀ နှင့် ၆၀ ပြည့်နှစ်များတွင် ဆီမီးကွန်ဒတ်တာ နည်းပညာ တိုးတက်လာမှုကြောင့် လက်တွေ့အသုံးချနိုင်သော Hall sensor များကို စီးပွားဖြစ် ထုတ်လုပ်နိုင်ခဲ့သည်။ ယနေ့ခေတ်တွင်မူ ၎င်းတို့ကို Microscopic အရွယ်အစားအထိ သေးငယ်အောင် ပြုလုပ်ပြီး Chip ပြားလေးများအတွင်း ထည့်သွင်းအသုံးပြုနေပြီဖြစ်သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Hall Effect sensor သည် **Lorentz Force** (သံလိုက်အားကြောင့် လျှပ်စစ်စီးဆင်းမှု လမ်းကြောင်းပြောင်းခြင်း) အပေါ် အခြေခံသည် -

* **လျှပ်စီးစီးဆင်းခြင်း-** Sensor အတွင်းရှိ လျှပ်ကူးပစ္စည်းပြားလေးထဲတွင် လျှပ်စစ်စီးကြောင်းတစ်ခုကို ပုံမှန်စီးဆင်းစေသည်။
* **သံလိုက်နှောင့်ယှက်ခြင်း-** သံလိုက်တစ်ခုကို Sensor နားသို့ ယူဆောင်လာသောအခါ သံလိုက်အားက စီးဆင်းနေသော အီလက်ထရွန်များကို ဘေးတစ်ဖက်သို့ တွန်းပို့လိုက်သည်။
* **ဗို့အားထွက်ပေါ်ခြင်း-** ထိုသို့ အီလက်ထရွန်များ တစ်ဖက်သို့ စုပုံသွားခြင်းကြောင့် လျှပ်ကူးပြား၏ ဘေးနှစ်ဖက်တွင် ဗို့အားကွာခြားမှု (**Hall Voltage**) ဖြစ်ပေါ်လာသည်။
* **အချက်ပြခြင်း-** ထိုဗို့အားလေးကို Circuit ဖြင့် မြှင့်တင်ပြီး Microcontroller က ဖတ်ယူနိုင်သော Signal အဖြစ် ထုတ်ပေးသည်။

---

### Hall Effect Sensor အမျိုးအစားများ
1. **Digital Hall Sensor:** ခလုတ်တစ်ခုကဲ့သို့ အလုပ်လုပ်သည်။ သံလိုက်ရှိလျှင် ON ပြီး မရှိလျှင် OFF ဖြစ်သည်။ (ဥပမာ - **A3144**)။
2. **Analog Hall Sensor:** သံလိုက်စက်ကွင်း အားကောင်းလျှင် ဗို့အားများများထုတ်ပေးပြီး အားနည်းလျှင် နည်းနည်းထုတ်ပေးသည်။ (ဥပမာ - **SS49E**)။
3. **Latching Hall Sensor:** သံလိုက်မြောက်ဝင်ရိုး (N) နဲ့ တွေ့ရင် ပွင့်သွားပြီး တောင်ဝင်ရိုး (S) နဲ့ တွေ့မှသာ ပြန်ပိတ်သည့် အမျိုးအစားဖြစ်သည်။

---

### အသုံးချပုံများ (Applications)
* **မော်တော်ကားလောက-** အင်ဂျင်၏ လည်ပတ်နှုန်း (**RPM**) ကို တိုင်းတာခြင်း။
* **စမတ်ဖုန်းများ-** ဖုန်းအဖုံး (Flip cover) ပိတ်လိုက်သည်နှင့် Screen အလိုအလျောက် ပိတ်သွားစေရန် သုံးခြင်း။
* **BLDC မော်တာများ-** မော်တာအတွင်းရှိ သံလိုက်ဝင်ရိုးများ၏ တည်နေရာကို သိရှိရန် သုံးခြင်း။
* **လုံခြုံရေး-** တံခါး သို့မဟုတ် ပြတင်းပေါက်များ ပွင့်နေခြင်း ရှိ၊ မရှိကို သံလိုက်လေးများကပ်၍ စစ်ဆေးခြင်း။
* **Current Sensing:** ဝါယာကြိုးကို ထိစရာမလိုဘဲ ၎င်းပတ်လည်ရှိ သံလိုက်စက်ကွင်းကို တိုင်းတာခြင်းဖြင့် လျှပ်စီးကြောင်း (**Amps**) မည်မျှစီးနေသည်ကို တွက်ချက်ခြင်း။
