## DIP Switch (PCB သုံး အသေးစားခလုတ်တွဲ)

### ခြုံငုံသုံးသပ်ချက် (Overview)
DIP Switch ဆိုသည်မှာ အီလက်ထရွန်းနစ်ပတ်လမ်းပြား (PCB) ပေါ်တွင် တပ်ဆင်အသုံးပြုရသော အသေးစားခလုတ်တွဲများ ဖြစ်ပါသည်။ "DIP" ဆိုသည့်အမည်မှာ ၎င်း၏အောက်ခြေရှိ ငုတ်တိုင်လေးများမှာ IC များကဲ့သို့ နှစ်တန်းအပြိုင် (**Dual In-line Package**) ပါရှိသောကြောင့် ခေါ်ဆိုခြင်းဖြစ်သည်။ ဤခလုတ်များကို စက်ပစ္စည်းတစ်ခု၏ Setting များ သို့မဟုတ် လုပ်ဆောင်ချက်ပုံစံများကို လက်ဖြင့်တစ်ခါတည်း သတ်မှတ်ချိန်ညှိထားရန် အသုံးပြုကြသည်။

---

### သမိုင်းကြောင်း (History)
DIP switch များသည် ၁၉၇၀ နှင့် ၈၀ ပြည့်နှစ်များ၊ ကွန်ပျူတာနည်းပညာဦးဦးဖျားဖျားခေတ်တွင် အလွန်ခေတ်စားခဲ့သည်။ ယနေ့ခေတ်ကဲ့သို့ Software မှတစ်ဆင့် Setting ချိန်၍မရသေးသောအချိန်က ကွန်ပျူတာ Motherboard များပေါ်တွင် Hardware လိပ်စာများ သတ်မှတ်ရန်နှင့် Memory ပမာဏချိန်ညှိရန် ၎င်းတို့ကို သုံးခဲ့ကြသည်။ Arcade ဂိမ်းစက်များတွင် ဂိမ်း၏ခက်ခဲမှုအဆင့် (Difficulty) နှင့် အကြွေစေ့စနစ်များကို ချိန်ညှိရာတွင်လည်း အကြီးအကျယ် အသုံးပြုခဲ့ကြသည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
DIP switch တစ်ခုသည် အခြေခံအားဖြင့် အသေးစားခလုတ်လေးများ (ပုံမှန်အားဖြင့် ၁ ခုမှ ၁၂ ခုအထိ) ကို အိမ်တစ်ခုတည်းတွင် စုစည်းထားခြင်းဖြစ်သည် -

[Image showing binary counting and configuration with a 4-position DIP switch]

* **Binary စနစ်-** ခလုတ်တစ်ခုချင်းစီသည် Binary ကိန်းဂဏန်းများဖြစ်သော ON (1) သို့မဟုတ် OFF (0) အခြေအနေကို ကိုယ်စားပြုသည်။
* **Setting သတ်မှတ်ခြင်း-** ခလုတ်လေးများကို အစီအစဉ်အလိုက် ပိတ်/ဖွင့် လုပ်ခြင်းဖြင့် Microcontroller ဆီသို့ သတ်မှတ်ထားသော ကုဒ် (Code) များ ပေးပို့ပြီး စက်၏လုပ်ဆောင်ချက်ကို ပြောင်းလဲစေသည်။
* **ယန္တရား-** ခလုတ်လေးကို တွန်းလိုက်သောအခါ အတွင်းရှိ သတ္တုပြားလေးက အောက်ခြေငုတ်နှစ်ခုကို ဆက်သွယ်ပေးလိုက်ခြင်းဖြင့် လျှပ်စစ်စီးဆင်းစေသည်။

---

### အသုံးများသော အမျိုးအစားများ
1. **Slide DIP Switch:** အသုံးအများဆုံးဖြစ်ပြီး ခလုတ်ခေါင်းလေးကို အပေါ်အောက် တွန်းရသော အမျိုးအစား။
2. **Piano DIP Switch:** စန္ဒရားခလုတ်များကဲ့သို့ ဘေးဘက်တွင် ခလုတ်များပါရှိသဖြင့် စက်အိမ်အတွင်းကျပ်နေသောနေရာများတွင် ချိန်ညှိရလွယ်ကူသည်။
3. **Rotary DIP Switch:** လီဗာများအစား ဂဏန်းများပါသည့် ဒိုင်ခွက်ကလေးကို လှည့်၍ Setting ချိန်ရသော အမျိုးအစား (ဥပမာ - ၀ မှ ၉ အထိ)။

### အသုံးချပုံများ (Applications)
* **Remote Control:** တံခါးအလိုအလျောက်ဖွင့်စက် (Garage door openers) များတွင် ကုဒ်နံပါတ်တူညီအောင် ချိန်ညှိခြင်း။
* **Industrial Boards:** စက်ရုံသုံးစက်ပစ္စည်းများတွင် အလုပ်လုပ်မည့် Mode ကို ရွေးချယ်ခြင်း။
* **Modems & Routers:** ယခင်က ဆက်သွယ်ရေးပစ္စည်းများတွင် လိုင်းခွဲသတ်မှတ်ရန် အသုံးပြုခဲ့သည်။
