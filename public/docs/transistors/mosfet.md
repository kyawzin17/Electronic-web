## MOSFET (Metal-Oxide-Semiconductor Field-Effect Transistor)

### ခြုံငုံသုံးသပ်ချက် (Overview)
MOSFET ဆိုသည်မှာ အချက်ပြလှိုင်းများကို မြှင့်တင်ရန်နှင့် ခလုတ်အဖြစ် အသုံးပြုရန် ထုတ်လုပ်ထားသော Field-Effect Transistor (FET) အမျိုးအစား ဖြစ်ပါသည်။ BJT နှင့် အဓိကကွာခြားချက်မှာ ၎င်းသည် **Voltage-controlled device** ဖြစ်ခြင်းပင် ဖြစ်သည်။ ဆိုလိုသည်မှာ Gate ငုတ်သို့ ပေးလိုက်သော "ဗို့အား" ဖြင့် Source နှင့် Drain ကြားရှိ လျှပ်စစ်စီးဆင်းမှုကို ထိန်းချုပ်ခြင်း ဖြစ်သည်။ ၎င်းသည် စွမ်းဆောင်ရည် အလွန်မြင့်မားသဖြင့် ယနေ့ခေတ် electronic ပစ္စည်းတိုင်းတွင် မပါမဖြစ် အသုံးပြုကြသည်။

---

### သမိုင်းကြောင်း (History)
၁၉၂၅ ခုနှစ်ကတည်းက သီအိုရီအရ တည်ရှိခဲ့သော်လည်း၊ အမှန်တကယ် အလုပ်လုပ်သော ပထမဆုံး MOSFET ကို ၁၉၅၉ ခုနှစ်တွင် Bell Labs မှ **Mohamed Atalla** နှင့် **Dawon Kahng** တို့က တီထွင်ခဲ့ခြင်း ဖြစ်သည်။ ဤတီထွင်မှုကြောင့် BJT များ၏ အပူထွက်ခြင်းနှင့် စွမ်းဆောင်ရည်ပိုင်း အားနည်းချက်များကို ကျော်လွှားနိုင်ခဲ့ပြီး ယနေ့ခေတ်သုံး CPU ပရိုဆက်ဆာများနှင့် Memory Chip များ ပေါ်ပေါက်လာစေခဲ့သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
MOSFET သည် "လျှပ်စစ်စက်ကွင်း" (**Electric Field**) ကို အသုံးချ၍ အလုပ်လုပ်ပါသည် -

* **အကာအကွယ်လွှာ-** Gate ငုတ်နှင့် ပတ်လမ်းကြားတွင် အလွန်ပါးလွှာသော Silicon Dioxide ($SiO_2$) အလွှာတစ်ခု ခြားထားသည်။ ထို့ကြောင့် Gate ထဲသို့ လျှပ်စစ်စီးဝင်ခြင်း မရှိသလောက် နည်းပါးသည်။
* **လျှပ်စစ်စက်ကွင်း-** Gate သို့ ဗို့အား ($V_{GS}$) ပေးလိုက်သောအခါ လျှပ်စစ်စက်ကွင်းတစ်ခု ဖြစ်ပေါ်လာသည်။
* **လမ်းကြောင်းဖြစ်ပေါ်ခြင်း-** ထိုစက်ကွင်းက Source နှင့် Drain ကြားတွင် လျှပ်စစ်စီးဆင်းနိုင်မည့် "**လမ်းကြောင်း**" (Channel) တစ်ခုကို ဖန်တီးပေးလိုက်သည်။

> **သဘောတရား-** Gate တွင် ဗို့အားပေးလျှင် လမ်းကြောင်းပွင့်သည် (ON)။ ဗို့အားမရှိလျှင် လမ်းကြောင်းပိတ်သည် (OFF)။

---

### အမျိုးအစားများ (Types)
1. **N-Channel (NMOS):** Gate ကို အဖိုဗို့အားပေးမှ ပွင့်သည်။ ပိုမြန်ပြီး ပိုအသုံးများသည်။
2. **P-Channel (PMOS):** Gate ကို ဗို့အားလျှော့ချပေးမှ (သို့မဟုတ် အနှုတ်ဗို့အားပေးမှ) ပွင့်သည်။
3. **Enhancement Mode:** ပုံမှန်အားဖြင့် ပိတ်နေပြီး ဗို့အားပေးမှ ပွင့်သော အမျိုးအစား (အသုံးအများဆုံး)။
4. **Depletion Mode:** ပုံမှန်အားဖြင့် ပွင့်နေပြီး ဗို့အားပေးမှ ပြန်ပိတ်သော အမျိုးအစား။

---

### အသုံးချပုံများ (Applications)
* **Power Electronics:** မော်တာများ၊ အပူပေးကိရိယာများနှင့် ပါဝါမြင့် LED များကို ဖွင့်/ပိတ် ပြုလုပ်ခြင်း။
* **Microprocessors:** CPU နှင့် GPU များအတွင်းရှိ logic ဂိတ်များအဖြစ် သန်းနှင့်ချီ၍ အသုံးပြုခြင်း။
* **Motor Controllers:** မော်တာ၏ အမြန်နှုန်းနှင့် လည်ပတ်မည့်ဘက်ကို ထိန်းချုပ်ပေးခြင်း။
* **Inverters:** ဘက်ထရီ သို့မဟုတ် ဆိုလာမှလာသော DC ကို အိမ်သုံး AC ဗို့အားသို့ ပြောင်းလဲပေးခြင်း။

---

### အခြေခံပုံသေနည်း (Key Formula)
MOSFET တစ်လုံးအတွက် အရေးအကြီးဆုံးမှာ **Threshold Voltage ($V_{GS(th)}$)** ဖြစ်ပြီး ၎င်းသည် ထရန်စစ္စတာ စတင်ပွင့်ရန် လိုအပ်သော အနည်းဆုံးဗို့အား ဖြစ်သည်။

Drain မှ စီးဆင်းမည့် လျှပ်စီးကြောင်း (**$I_D$**):

&emsp; **$$I_D = K(V_{GS} - V_{GS(th)})^2$$**

**ဤနေရာတွင်:**
* $I_D$ = Drain Current
* $V_{GS}$ = Gate-to-Source Voltage
* $K$ = MOSFET ၏ တည်ဆောက်ပုံအပေါ် မူတည်သော ကိန်းသေ (Conductance parameter)
