## Flip-Flop (ဒစ်ဂျစ်တယ် မှတ်ဉာဏ်ယူနစ်)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Flip-Flop ဆိုသည်မှာ ဒစ်ဂျစ်တယ် အချက်အလက် ၁-ဘစ် (1 bit of data) ကို မှတ်သားထားနိုင်သော အခြေခံအကျဆုံး "**မှတ်ဉာဏ်ပတ်လမ်း**" ဖြစ်ပါသည်။ သာမန် Logic Gate များသည် လက်ရှိပေးလိုက်သော Input အပေါ်တွင်သာ မူတည်၍ အလုပ်လုပ်သော်လည်း Flip-Flop သည် လက်ရှိ Input အပြင် ယခင်က ရှိခဲ့သော အခြေအနေ (Previous State) အပေါ်တွင်ပါ မူတည်၍ ဆုံးဖြတ်ချက်ချပါသည်။ ထို့ကြောင့် ၎င်းကို ကွန်ပျူတာ၏ မှတ်ဉာဏ် (Memory)၊ Register များနှင့် ကောင်တာ (Counter) များ တည်ဆောက်ရာတွင် အဓိက အသုံးပြုသည်။

---

### သမိုင်းကြောင်း (History)
ပထမဆုံးသော Flip-Flop ကို ၁၉၁၈ ခုနှစ်တွင် **William Eccles** နှင့် **F. W. Jordan** တို့က Vacuum tube များ သုံးပြီး တည်ဆောက်ခဲ့ကြသည်။ ထိုစဉ်က ၎င်းကို **Eccles–Jordan trigger circuit** ဟု ခေါ်ခဲ့သည်။ ၁၉၆၀ ပြည့်နှစ်များတွင် IC နည်းပညာ ပေါ်ထွက်လာသောအခါ **7474 (D Flip-Flop)** နှင့် **7476 (JK Flip-Flop)** ကဲ့သို့သော IC များသည် ကွန်ပျူတာ၏ RAM နှင့် CPU များအတွင်း အချက်အလက်များ သိမ်းဆည်းရန်အတွက် မရှိမဖြစ် အစိတ်အပိုင်းများ ဖြစ်လာခဲ့သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Flip-Flop သည် တုံ့ပြန်မှုစနစ် (**Feedback**) အပေါ် အခြေခံ၍ အလုပ်လုပ်သည် -

* **တည်ငြိမ်သောအခြေအနေ-** ၎င်းသည် လျှပ်စစ်ဓာတ်အား ရရှိနေသရွေ့ "၁" သို့မဟုတ် "၀" အခြေအနေတစ်ခုခုမှာ အမြဲတမ်း တည်ငြိမ်စွာ ရှိနေနိုင်သည်။
* **နာရီချက်လှိုင်း (Clocking):** ခေတ်ပေါ် Flip-Flop အများစုသည် Clock Pulse တစ်ခု၏ အတက်လှိုင်း (Rising Edge) သို့မဟုတ် အဆင်းလှိုင်း (Falling Edge) ရောက်မှသာ အလုပ်လုပ်ကြသည်။
* **မှတ်သားခြင်း (Latching):** အချက်အလက်တစ်ခုကို လက်ခံရရှိပြီးလျှင် နောက်ထပ် အမိန့်အသစ် မလာမချင်း ထိုအချက်အလက်ကို အထွက် (Output) မှာ ဆွဲကိုင်ထား (Latch) ပေးထားသည်။

---

### အသုံးများသော Flip-Flop အမျိုးအစားများ
1. **SR Flip-Flop (Set-Reset):** အခြေခံအကျဆုံး အမျိုးအစား။ Set ငုတ်က ၁ ဖြစ်စေပြီး Reset ငုတ်က ၀ ဖြစ်စေသည်။
2. **D Flip-Flop (Data/Delay):** Clock ရိုက်လိုက်ချိန်တွင် အဝင်ရှိ အချက်အလက်ကို အထွက်သို့ ပို့ပေးပြီး မှတ်သားထားသည်။ Register များတွင် အသုံးအများဆုံးဖြစ်သည်။
3. **JK Flip-Flop:** SR ၏ အားနည်းချက် (Invalid State) ကို ပြုပြင်ထားသော အမျိုးအစား။ Input နှစ်ခုလုံး ၁ ပေးလျှင် ယခင်ရှိနေသော အခြေအနေကို ပြောင်းပြန် (**Toggle**) လှန်ပေးသည်။
4. **T Flip-Flop (Toggle):** အချက်ပြလှိုင်း ရောက်လာတိုင်း အခြေအနေကို ပြောင်းပြန်လှန်ပေးသည်။ Frequency Division များတွင် သုံးသည်။

### အသုံးချပုံများ (Applications)
* **ဒစ်ဂျစ်တယ် မှတ်ဉာဏ်-** Flip-Flop (၈) ခုကို စုပေါင်းလိုက်လျှင် ၁-ဘိုက် (1 Byte) သိမ်းနိုင်သော Register တစ်ခု ရရှိသည်။ ၎င်းတို့ကို သန်းပေါင်းများစွာ စုစည်း၍ RAM များ တည်ဆောက်သည်။
* **ကောင်တာများ (Counters):** ဒစ်ဂျစ်တယ် နာရီများနှင့် တိုင်မာများတွင် နံပါတ်များ ရေတွက်ရန် သုံးသည်။
* **လှိုင်းအကြိမ်ရေ လျှော့ချခြင်း-** မြန်နှုန်းမြင့်လှိုင်းများကို ပိုမိုနှေးကွေးသော လှိုင်းများအဖြစ်သို့ ခွဲဝေထုတ်ပေးခြင်း (Frequency Division)။
* **အချက်အလက် ထိန်းညှိခြင်း-** ဒေတာများကို ပတ်လမ်းအတွင်း တည်ငြိမ်စွာ စီးဆင်းနိုင်အောင် ထိန်းညှိပေးခြင်း။
