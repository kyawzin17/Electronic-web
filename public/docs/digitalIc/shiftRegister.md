## Shift Register (ဒစ်ဂျစ်တယ် အချက်အလက် လွှဲပြောင်းစနစ်)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Shift Register ဆိုသည်မှာ ဒစ်ဂျစ်တယ် အချက်အလက်များကို သိမ်းဆည်းရန်နှင့် လွှဲပြောင်းပေးပို့ရန် အသုံးပြုသော ပတ်လမ်းတစ်ခု ဖြစ်ပါသည်။ ၎င်းကို Flip-Flop များ (အများအားဖြင့် D Flip-Flop) ကို စီးရီးချက်ဆက်၍ တည်ဆောက်ထားသည်။ တစ်ခု၏ Output သည် နောက်တစ်ခု၏ Input ဖြစ်နေပြီး၊ နာရီချက်လှိုင်း (Clock pulse) တစ်ချက် ရောက်လာတိုင်း အချက်အလက် (Bits) များသည် ရှေ့မှနောက်သို့ တစ်နေရာစီ "ရွှေ့လျား" (Shift) သွားကြသည်။

---

### သမိုင်းကြောင်း (History)
ဒေတာများကို ရွှေ့ပြောင်းခြင်း အယူအဆသည် အစောပိုင်း ကွန်ပျူတာများ၏ မြှောက်ခြင်းနှင့် စားခြင်း တွက်ချက်မှုများအတွက် အလွန်အရေးပါခဲ့သည်။ ၁၉၄၀ ပြည့်နှစ်များတွင် Vacuum tube များကို သုံး၍ တည်ဆောက်ခဲ့ကြသော်လည်း ၁၉၇၀ ပြည့်နှစ်များတွင် **74164** နှင့် **74165** ကဲ့သို့သော IC များ ပေါ်ထွက်လာခဲ့သည်။ ဤ IC များသည် ကွန်ပျူတာနှင့် ပြင်ပပစ္စည်းများ (ဥပမာ - Keyboard နှင့် Printer) ကြား ဆက်သွယ်ရာတွင် ကြိုးလိုင်းအနည်းငယ်ဖြင့် မြန်မြန်ဆန်ဆန် ဆက်သွယ်နိုင်အောင် ကူညီပေးခဲ့သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Shift Register သည် Clock Pulse နှင့် အချိန်ကိုက် အလုပ်လုပ်သည် -

* **ဒေတာရွှေ့လျားခြင်း-** Clock pulse တစ်ချက် ရောက်လာလျှင် ပထမ Flip-flop ထဲရှိ Data သည် ဒုတိယတစ်ခုသို့၊ ဒုတိယမှ တတိယသို့ ဆင့်ကဲ ရွှေ့သွားသည်။
* **လားရာ-** အချက်အလက်များကို ညာဘက်သို့ ရွှေ့ခြင်း (**Shift-Right**) သို့မဟုတ် ဘယ်ဘက်သို့ ရွှေ့ခြင်း (**Shift-Left**) ပြုလုပ်နိုင်သည်။
* **မှတ်သားထားခြင်း-** ဒေတာများကို ရွှေ့နေစဉ်အတွင်း ၎င်းသည် ခေတ္တမှတ်ဉာဏ် (Temporary Memory) အဖြစ်လည်း လုပ်ဆောင်ပေးသည်။

### Shift Register အမျိုးအစားများ
1. **SISO (Serial-In, Serial-Out):** အချက်အလက် တစ်ခုချင်းဝင်ပြီး တစ်ခုချင်း ပြန်ထွက်သည်။ Signal များကို အချိန်ဆိုင်း (Delay) ရန် သုံးသည်။
2. **SIPO (Serial-In, Parallel-Out):** အချက်အလက် တစ်ခုချင်းဝင်သော်လည်း အထွက်တွင် အားလုံးတစ်ပြိုင်နက် ထွက်လာသည်။ Arduino ငုတ်အနည်းငယ်ဖြင့် LED များစွာကို ထိန်းချုပ်ရာတွင် သုံးသည် (ဥပမာ - **74HC595 IC**)။
3. **PISO (Parallel-In, Serial-Out):** အချက်အလက်များကို တစ်ပြိုင်နက် ထည့်သွင်းပြီး အပြင်သို့ တစ်ခုချင်းစီ ပြန်ထုတ်ပေးသည်။ ခလုတ်များစွာ၏ အခြေအနေကို ကြိုးတစ်ပင်တည်းဖြင့် ဖတ်ရန် သုံးသည် (ဥပမာ - **74HC165 IC**)။
4. **PIPO (Parallel-In, Parallel-Out):** အချက်အလက်များကို တစ်ပြိုင်နက်သွင်းပြီး တစ်ပြိုင်နက် ပြန်ထုတ်သည်။ ဒေတာများကို ခေတ္တသိမ်းထားရန် သုံးသည်။

### အသုံးချပုံများ (Applications)
* **I/O Expansion:** Arduino ၏ ငုတ် ၃ ခုတည်းဖြင့် LED ၈ လုံး သို့မဟုတ် ထိုထက်မကကို ထိန်းချုပ်ခြင်း (74HC595 ကို သုံး၍)။
* **ဒေတာပုံစံ ပြောင်းလဲခြင်း-** Serial data များကို Parallel data အဖြစ် ပြောင်းလဲပြီး CPU က ဖတ်နိုင်အောင် ပြုလုပ်ခြင်း။
* **သင်္ချာတွက်ချက်မှုများ-** Binary ကိန်းဂဏန်းများကို ဘယ်ဘက်သို့ တစ်ကြိမ်ရွှေ့လျှင် ၂ နှင့် မြှောက်ခြင်းဖြစ်ပြီး၊ ညာဘက်သို့ တစ်ကြိမ်ရွှေ့လျှင် ၂ နှင့် စားခြင်း ဖြစ်သည်။
* **Digital Delay:** အချက်ပြလှိုင်းများကို သတ်မှတ်ထားသော အချိန်တစ်ခုအထိ စောင့်ဆိုင်းစေခြင်း။
