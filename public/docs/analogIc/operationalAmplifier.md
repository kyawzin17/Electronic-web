## Operational Amplifier (Op-Amp)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Operational Amplifier (Op-Amp) ဆိုသည်မှာ အဝင်ငုတ်နှစ်ခု၏ ဗို့အားခြားနားချက်ကို အဆပေါင်းများစွာ မြှင့်တင်ပေးနိုင်သော **High-gain voltage amplifier** ဖြစ်ပါသည်။ ၎င်းကို မူလက ပေါင်းခြင်း၊ နှုတ်ခြင်း၊ အိတ်သီဂရယ်ယူခြင်း (Integration) စသည့် သင်္ချာတွက်ချက်မှုများ ပြုလုပ်ရန် ဒီဇိုင်းထုတ်ခဲ့သော်လည်း ယနေ့ခေတ်တွင် Analog electronic ပတ်လမ်းများ၏ အခြေခံအကျဆုံး အစိတ်အပိုင်းတစ်ခုအဖြစ် နယ်ပယ်ပေါင်းစုံ၌ အသုံးပြုနေကြပြီ ဖြစ်သည်။

### သမိုင်းကြောင်း (History)
Op-Amp များကို ၁၉၄၀ ပြည့်နှစ်များတွင် Vacuum tubes များဖြင့် စတင်တည်ဆောက်ခဲ့ပြီး စစ်ဘက်သုံး Analog ကွန်ပျူတာများတွင် အသုံးပြုခဲ့သည်။ ၁၉၆၃ ခုနှစ်တွင် Bob Widlar က ပထမဆုံးသော Chip တစ်ခုတည်းပါ Op-Amp ကို ဖန်တီးခဲ့သည်။ သို့သော် ၁၉၆၈ ခုနှစ်တွင် ထွက်ရှိလာသော **$\mu$A741** သည် အောင်မြင်မှုအရှိဆုံး ဖြစ်ခဲ့ပြီး အီလက်ထရွန်းနစ်လောကတွင် Op-Amp များကို နေရာတိုင်း၌ သုံးစွဲနိုင်ရန် လမ်းစဖွင့်ပေးခဲ့သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Op-Amp သည် ၎င်း၏ Input ငုတ်နှစ်ခုကြားရှိ ဗို့အားကွာဟချက်ကို အခြေခံ၍ အလုပ်လုပ်သည် -

* **Inverting Input (-):** ဤငုတ်သို့ အချက်ပြလှိုင်းပေးပါက အထွက် (Output) တွင် လှိုင်းမှာ ပြောင်းပြန် (Phase ၁၈၀ ဒီဂရီ ပြောင်းပြီး) ထွက်လာမည်။
* **Non-Inverting Input (+):** ဤငုတ်သို့ အချက်ပြလှိုင်းပေးပါက အထွက်တွင် လှိုင်းမှာ မူလအတိုင်း (Phase မပြောင်းဘဲ) ထွက်လာမည်။
* **ထိန်းချုပ်ပုံ:** Negative Feedback ပေးထားသော ပတ်လမ်းများတွင် Op-Amp သည် Input ငုတ်နှစ်ခုလုံး၏ ဗို့အားတူညီနေစေရန် (Virtual Ground Concept) အမြဲတမ်း ကြိုးစားထိန်းညှိပေးသည်။

> **အခြေခံစည်းမျဉ်းများ:**
> * Input ငုတ်များထဲသို့ လျှပ်စစ်စီးဝင်ခြင်း မရှိသလောက် နည်းပါးသည် (**High Input Impedance** IA)။
> * Output သည် Input ငုတ်နှစ်ခုကြားရှိ ဗို့အားခြားနားချက်ကို သုညဖြစ်အောင် အမြဲတမ်း ချိန်ညှိပေးသည်။

---

### အမျိုးအစားများ (Types)
1. **LM741:** အခြေခံအကျဆုံးနှင့် အသုံးအများဆုံး Op-Amp (၈ ငုတ်ပါ IC)။
2. **LM358:** ပါဝါလိုင်းတစ်ခုတည်း (Single supply) နှင့် အလုပ်လုပ်နိုင်သော Op-Amp နှစ်လုံးတွဲ။
3. **TL072:** အသံပိုင်းဆိုင်ရာ (Audio) အတွက် ဆူညံသံနည်းပါးသော JFET-input အမျိုးအစား။
4. **NE5532:** ပရော်ဖက်ရှင်နယ် အသံချဲ့စက်များတွင် သုံးသော စွမ်းဆောင်ရည်မြင့် အမျိုးအစား။

---

### အသုံးချပုံများ (Applications)
* **Signal Amplification-** မိုက်ခရိုဖုန်း သို့မဟုတ် Sensor များမှလာသော အားနည်းသည့် လျှပ်စစ်လှိုင်းများကို မြှင့်တင်ပေးခြင်း။
* **Active Filters-** လိုချင်သော လှိုင်းအကြိမ်ရေ (Frequency) ကိုသာ ရွေးချယ်ဖြတ်သန်းစေသည့် Low-pass/High-pass filter များ တည်ဆောက်ခြင်း။
* **Comparators-** ဗို့အားနှစ်ခုကို နှိုင်းယှဉ်ပြီး မည်သည့်ဘက်က ပိုများသလဲဆိုသည်ကို စစ်ဆေးခြင်း။
* **သင်္ချာတွက်ချက်မှုများ-** Mixer ပတ်လမ်းများတွင် အချက်ပြလှိုင်းများစွာကို ပေါင်းခြင်း သို့မဟုတ် နှုတ်ခြင်း။
* **Voltage Follower-** အချက်ပြလှိုင်းတစ်ခုကို ပုံစံမပျက်စေဘဲ (Buffer အဖြစ်) နောက်ပတ်လမ်းတစ်ခုသို့ ကူးပြောင်းပေးခြင်း။

---

### အခြေခံပုံသေနည်း (Key Formula)

**Inverting Amplifier** အတွက် မြှင့်တင်မှုနှုန်း (Gain - $A_v$):

&emsp; **$$A_v = -\frac{R_f}{R_{in}}$$**

**Non-Inverting Amplifier** အတွက် မြှင့်တင်မှုနှုန်း (Gain - $A_v$):

&emsp; **$$A_v = 1 + \frac{R_f}{R_1}$$**
