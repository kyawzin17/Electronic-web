## SRAM (မြန်နှုန်းမြင့် တည်ငြိမ်မှတ်ဉာဏ်)

### ခြုံငုံသုံးသပ်ချက် (Overview)
SRAM (Static Random Access Memory) ဆိုသည်မှာ Flip-flop ပတ်လမ်းများကို အသုံးပြု၍ ဒေတာများကို သိမ်းဆည်းသော Semiconductor မှတ်ဉာဏ်အမျိုးအစား ဖြစ်ပါသည်။ "Static" ဟု ခေါ်ဆိုရခြင်းမှာ လျှပ်စစ်ဓာတ်အား ရှိနေသရွေ့ ဒေတာများကို ပုံသေမှတ်သားထားနိုင်သောကြောင့် ဖြစ်သည်။ DRAM (Dynamic RAM) ကဲ့သို့ ဒေတာများ မပျောက်ပျက်စေရန် ခဏခဏ ပြန်လည်ဖြည့်တင်းပေးရန် (Refresh လုပ်ရန်) မလိုပါ။ SRAM သည် အလွန်မြန်ဆန်သော အရှိန်ရှိသော်လည်း ထုတ်လုပ်မှုစရိတ် ကြီးမြင့်ပြီး နေရာယူမှု များပြားသည်။

---

### သမိုင်းကြောင်း (History)
ပထမဆုံး SRAM ကို ၁၉၆၃ ခုနှစ်တွင် Fairchild Semiconductor မှ Robert Norman က တီထွင်ခဲ့သည်။ ၁၉၆၉ ခုနှစ်တွင် Intel ကုမ္ပဏီ၏ ပထမဆုံးထုတ်ကုန်ဖြစ်သော **3101 (64-bit SRAM chip)** ကို ထုတ်လုပ်ခဲ့သည်။ ဆယ်စုနှစ်များစွာအတွင်း SRAM သည် ရိုးရှင်းသော မှတ်ဉာဏ်အဆင့်မှသည် ယနေ့ခေတ် CPU များ၏ အမြန်နှုန်းကို ထိန်းညှိပေးသည့် **Cache Memory** အဆင့်အထိ တိုးတက်လာခဲ့သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
SRAM သည် ဒေတာများကို ထရန်စစ္စတာများ ပေါင်းစပ်ဖွဲ့စည်းမှု (ပုံမှန်အားဖြင့် Memory Cell တစ်ခုလျှင် ထရန်စစ္စတာ ၆ လုံး - **6T SRAM**) ဖြင့် သိမ်းဆည်းသည် -

* **Flip-Flop စနစ်-** Memory cell တစ်ခုစီကို အပြန်အလှန် ချိတ်ဆက်ထားသော Inverter နှစ်ခုဖြင့် တည်ဆောက်ထားသည်။ ၎င်းသည် Binary 0 သို့မဟုတ် 1 အခြေအနေကို ခိုင်မြဲစွာ မှတ်သားထားနိုင်သည်။
* **Refresh လုပ်ရန် မလိုခြင်း-** DRAM ကဲ့သို့ လျှပ်သို (Capacitor) ကို မသုံးဘဲ ထရန်စစ္စတာများကိုသာ သုံးသဖြင့် လျှပ်စစ်ဓာတ်အား ယိုစိမ့်မှု မရှိသလောက် နည်းပါးသည်။ ထို့ကြောင့် Refresh လုပ်ရန် အချိန်ပေးစရာ မလိုဘဲ တိုက်ရိုက် အလုပ်လုပ်နိုင်သည်။
* **အမြန်နှုန်း-** SRAM ၏ ဒေတာဖတ်ယူမှု အမြန်နှုန်းမှာ အလွန်မြင့်မားပြီး ပုံမှန်အားဖြင့် $1\text{ မှ }10\text{ nanoseconds}$ အတွင်းသာ ကြာမြင့်သည်။

---

### SRAM အမျိုးအစားများ
1. **Asynchronous SRAM:** စနစ်၏ နာရီချက်လှိုင်း (System Clock) ပေါ်တွင် မမှီခိုဘဲ သီးခြားအလုပ်လုပ်သည်။ ရိုးရှင်းသော အီလက်ထရွန်းနစ် ပစ္စည်းများတွင် သုံးသည်။
2. **Synchronous SRAM (SSRAM):** လုပ်ဆောင်ချက်အားလုံးကို System Clock နှင့် ချိတ်ဆက်ထားသဖြင့် ဒေတာပေးပို့မှု ပိုမိုမြန်ဆန်သည်။
3. **Low-Power SRAM:** ဘက်ထရီဖြင့် အလုပ်လုပ်သော ပစ္စည်းများအတွက် ပါဝါစားသုံးမှု အနည်းဆုံးဖြစ်အောင် ထုတ်လုပ်ထားသည်။

---

### အသုံးချပုံများ (Applications)
* **CPU Cache:** ပရိုဆက်ဆာများ၏ L1, L2, L3 cache များအဖြစ် သုံးပြီး CPU ၏ အမြန်နှုန်းကို မြှင့်တင်ပေးခြင်း။
* **Microcontrollers:** မြန်ဆန်သော တွက်ချက်မှုများ လိုအပ်သည့် Chip များအတွင်းရှိ Internal RAM အဖြစ် သုံးခြင်း။
* **ဆက်သွယ်ရေးစက်ပစ္စည်းများ-** မြန်နှုန်းမြင့် Router နှင့် Switch များတွင် Buffer memory အဖြစ် သုံးခြင်း။
* **Buffer Memory:** အသံပိုင်းဆိုင်ရာ စက်ပစ္စည်းများနှင့် High-end DAC များတွင် ဒေတာများကို ခေတ္တထိန်းသိမ်းရန် သုံးခြင်း။
