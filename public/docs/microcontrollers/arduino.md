## Arduino (အာဒွီနို အီလက်ထရွန်းနစ် ပလက်ဖောင်း)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Arduino ဆိုသည်မှာ အသုံးပြုရလွယ်ကူပြီး ပျော့ပြောင်းမှုရှိသော Hardware နှင့် Software များအပေါ် အခြေခံထားသည့် **Open-source** အီလက်ထရွန်းနစ် ပလက်ဖောင်းတစ်ခု ဖြစ်ပါသည်။ ၎င်းတွင် ပရိုဂရမ်သွင်းနိုင်သော ပတ်လမ်းပြား (Microcontroller Board) နှင့် ကွန်ပျူတာပေါ်တွင် ကုဒ်ရေးသားရန် အသုံးပြုသည့် Software (**IDE**) တို့ ပါဝင်သည်။ အနုပညာရှင်များ၊ ဒီဇိုင်နာများ၊ ဝါသနာရှင်များနှင့် အီလက်ထရွန်းနစ်ကို စိတ်ဝင်စားသူ မည်သူမဆို အပြန်အလှန်တုံ့ပြန်မှုရှိသော (Interactive) ပစ္စည်းများကို အလွယ်တကူ တည်ဆောက်နိုင်ရန် ရည်ရွယ်ထုတ်လုပ်ထားခြင်း ဖြစ်သည်။

---

### သမိုင်းကြောင်း (History)
Arduino ကို ၂၀၀၅ ခုနှစ်တွင် အီတလီနိုင်ငံရှိ Ivrea Interaction Design Institute ၌ စတင်ဖန်တီးခဲ့သည်။ တည်ထောင်သူများမှာ **Massimo Banzi** နှင့် အဖွဲ့ဝင်များ ဖြစ်ကြသည်။ ထိုခေတ်က အီလက်ထရွန်းနစ် သင်ကြားရေးပစ္စည်းများမှာ ဈေးကြီးပြီး သင်ယူရခက်ခဲသောကြောင့် ကျောင်းသားများအတွက် ဈေးသက်သာပြီး လွယ်ကူသော ကိရိယာတစ်ခုအဖြစ် ဖန်တီးခဲ့ခြင်း ဖြစ်သည်။ "Arduino" ဟူသော အမည်မှာ ၎င်းတို့အဖွဲ့ အမြဲဆုံတွေ့လေ့ရှိသည့် အီတလီရှိ ဘား (Bar) တစ်ခု၏ အမည်ကို အစွဲပြု၍ ပေးထားခြင်း ဖြစ်သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Arduino သည် ရိုးရှင်းသော **Sense-Think-Act** ပုံစံဖြင့် အလုပ်လုပ်သည် -

* **Sense (အာရုံခံခြင်း):** အာရုံခံကိရိယာ (Sensors) များမှတစ်ဆင့် အလင်း၊ အပူချိန် သို့မဟုတ် ခလုတ်နှိပ်ခြင်း စသည့် အချက်အလက်များကို လက်ခံသည်။
* **Think (စဉ်းစားခြင်း):** သင်ရေးသားထားသော ကုဒ် (**Sketch**) အပေါ် မူတည်၍ Microcontroller က ထိုအချက်အလက်များကို တွက်ချက်သည်။
* **Act (တုံ့ပြန်ခြင်း):** ရရှိလာသော ရလဒ်အတိုင်း မော်တာလှည့်ခြင်း၊ မီးလင်းခြင်း စသည့် လုပ်ဆောင်ချက်များကို အထွက် (Output) အဖြစ် လုပ်ဆောင်ပေးသည်။

---

### အသုံးများသော Arduino Board များ
1. **Arduino UNO R3:** အခြေခံလေ့လာသူများအတွက် အသင့်တော်ဆုံးနှင့် အသုံးအများဆုံး Board ဖြစ်သည်။
2. **Arduino Nano:** အရွယ်အစား သေးငယ်ပြီး Breadboard ပေါ်တွင် တိုက်ရိုက်စိုက်၍ သုံးနိုင်သည်။
3. **Arduino Mega 2560:** မှတ်ဉာဏ် (Memory) နှင့် ငုတ် (Pins) များစွာ လိုအပ်သော ပရောဂျက်ကြီးများအတွက် သုံးသည်။
4. **Arduino Leonardo:** ကွန်ပျူတာ၏ Keyboard သို့မဟုတ် Mouse ကဲ့သို့ အလုပ်လုပ်အောင် ဖန်တီးနိုင်သည်။

### အသုံးချပုံများ (Applications)
* **Smart Home:** အိမ်တွင်းရှိ မီးများ၊ ဖန်ကာများကို ဖုန်းဖြင့် ထိန်းချုပ်ခြင်း။
* **Robotics:** စက်ရုပ်လက်တံများ၊ လမ်းကြောင်းလိုက်စက်ရုပ်များနှင့် ဒရုန်းများ တည်ဆောက်ခြင်း။
* **IoT (Internet of Things):** အပူချိန်နှင့် စိုထိုင်းဆများကို အင်တာနက်ပေါ်သို့ ပို့ဆောင် သိမ်းဆည်းခြင်း။
* **စိုက်ပျိုးရေး-** မြေဆီလွှာ၏ စိုထိုင်းဆကို တိုင်းတာပြီး အလိုအလျောက် ရေလောင်းသည့် စနစ်များ ဖန်တီးခြင်း။
