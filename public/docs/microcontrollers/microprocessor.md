## Microprocessor (မိုက်ခရိုပရိုဆက်ဆာ)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Microprocessor ဆိုသည်မှာ ဒစ်ဂျစ်တယ်စနစ်တစ်ခု၏ "**ဦးနှောက်**" ဖြစ်ပြီး Binary data များကို အဝင် (Input) အဖြစ် လက်ခံကာ၊ Memory ထဲတွင် သိမ်းဆည်းထားသော ညွှန်ကြားချက်များ (Instructions) အတိုင်း တွက်ချက်လုပ်ဆောင်ပြီး ရလဒ်များကို အထွက် (Output) အဖြစ် ထုတ်ပေးသည့် ပရိုဂရမ်သွင်းနိုင်သော IC တစ်ခု ဖြစ်ပါသည်။ ၎င်းသည် Central Processing Unit (CPU) ၏ လုပ်ဆောင်ချက်အားလုံးကို Chip ပြားလေးတစ်ခုတည်းပေါ်တွင် စုစည်းထားခြင်း ဖြစ်သည်။

---

### သမိုင်းကြောင်း (History)
မိုက်ခရိုပရိုဆက်ဆာ တော်လှန်ရေးသည် ၁၉၇၁ ခုနှစ်တွင် Intel မှ ကမ္ဘာ့ပထမဆုံး တစ်လုံးတည်းသော Chip ဖြစ်သည့် **4004** ကို ထုတ်လုပ်ရာမှ စတင်ခဲ့သည်။ မူလက ဂဏန်းတွက်စက်များအတွက် ရည်ရွယ်ခဲ့သော်လည်း နောက်ပိုင်းတွင် 8080 နှင့် ယနေ့ခေတ် PC များတွင် သုံးနေသော x86 architecture များအထိ တိုးတက်လာခဲ့သည်။ ဆယ်စုနှစ်များစွာအတွင်း ပရိုဆက်ဆာအတွင်းရှိ ထရန်စစ္စတာ အရေအတွက်သည် ထောင်ဂဏန်းမှသည် ဘီလီယံပေါင်းများစွာအထိ တိုးပွားလာပြီး အလုပ်လုပ်နှုန်းမှာလည်း အဆပေါင်းများစွာ မြန်ဆန်လာခဲ့သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Microprocessor သည် **Instruction Cycle** ဟုခေါ်သော အဆင့် (၄) ဆင့်ဖြင့် ထပ်ခါတလဲလဲ အလုပ်လုပ်သည် -

* **Fetch (ယူဆောင်ခြင်း):** Memory ထဲမှ လုပ်ဆောင်ရမည့် ညွှန်ကြားချက်ကို ယူဆောင်သည်။
* **Decode (အဓိပ္ပာယ်ဖော်ခြင်း):** Control Unit (CU) က ထိုညွှန်ကြားချက်သည် ဘာလုပ်ခိုင်းခြင်းဖြစ်သည် (ပေါင်းခိုင်းခြင်းလား၊ ရွှေ့ခိုင်းခြင်းလား) ကို အဓိပ္ပာယ်ဖော်သည်။
* **Execute (လုပ်ဆောင်ခြင်း):** ALU က အမှန်တကယ် တွက်ချက်ခြင်း သို့မဟုတ် ဆုံးဖြတ်ခြင်းကို လုပ်ဆောင်သည်။
* **Store (သိမ်းဆည်းခြင်း):** ရရှိလာသော ရလဒ်ကို Register သို့မဟုတ် Memory ထဲသို့ ပြန်လည်သိမ်းဆည်းသည်။

---

### အတွင်းပိုင်း အစိတ်အပိုင်းများ (Internal Components)
* **ALU (Arithmetic Logic Unit):** သင်္ချာဆိုင်ရာ တွက်ချက်မှုများနှင့် Logic ဆိုင်ရာ ဆုံးဖြတ်ချက်များကို လုပ်ဆောင်ပေးသည်။
* **Control Unit (CU):** ပရိုဆက်ဆာ၏ အစိတ်အပိုင်းအားလုံးကို ထိန်းချုပ်ပြီး ဒေတာစီးဆင်းမှုကို စီမံခန့်ခွဲပေးသည်။
* **Registers:** တွက်ချက်နေစဉ်အတွင်း ဒေတာများကို ခေတ္တသိမ်းဆည်းထားသည့် အလွန်မြန်သော နေရာများ။
* **Cache Memory:** မကြာခဏ သုံးလေ့ရှိသော ဒေတာများကို သိမ်းဆည်းထားသည့် ပရိုဆက်ဆာအတွင်းရှိ အမြန်ဆုံးမှတ်ဉာဏ်။
* **Buses:** ဒေတာများ၊ လိပ်စာများနှင့် ထိန်းချုပ်မှု လှိုင်းများကို သယ်ဆောင်ပေးသည့် လမ်းကြောင်းများ။

---

### အသုံးချပုံများ (Applications)
* **ကိုယ်ပိုင်ကွန်ပျူတာများ-** Laptop နှင့် Desktop များ၏ အဓိက အသက်ဝိညာဉ် (ဥပမာ - Intel Core, AMD Ryzen)။
* **လက်ကိုင်ဖုန်းများ-** Smartphone များနှင့် Tablet များအတွင်းရှိ CPU များ။
* **စက်မှုလုပ်ငန်းများ-** စက်ရုပ်များနှင့် အလိုအလျောက် ထုတ်လုပ်ရေးလိုင်းများကို ထိန်းချုပ်ခြင်း။
* **Embedded Systems:** ကားအင်ဂျင် ထိန်းချုပ်စနစ်များ၊ ဆေးဘက်ဆိုင်ရာ ကိရိယာများနှင့် ဆက်သွယ်ရေး စက်ပစ္စည်းများ။

---

### အခြေခံပုံသေနည်း (စွမ်းဆောင်ရည်)
ပရိုဆက်ဆာတစ်ခု၏ စွမ်းဆောင်ရည်ကို ညွှန်ကြားချက်တစ်ခု လုပ်ဆောင်ရန် ကြာမြင့်ချိန်ဖြင့် တိုင်းတာလေ့ရှိသည် -

$$Execution\ Time = \frac{Instructions \times CPI}{Frequency}$$

> *(CPI = Cycles Per Instruction)*
