## Microcontroller (မိုက်ခရိုကွန်ထရိုလာ)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Microcontroller ဆိုသည်မှာ Chip ပြားလေးတစ်ခုတည်းပေါ်တွင် Processor၊ Memory နှင့် Input/Output ငုတ်များအားလုံး စုစည်းပါဝင်နေသော "ကွန်ပျူတာအသေးစား" (Computer-on-a-chip) ဖြစ်ပါသည်။ ၎င်းကို သီးခြားလုပ်ဆောင်ချက်တစ်ခု (Specific Task) ကို အလိုအလျောက် လုပ်ဆောင်နိုင်ရန် ပရိုဂရမ်ရေးသား၍ အသုံးပြုကြသည်။ သာမန် Desktop ကွန်ပျူတာများကဲ့သို့ ရှုပ်ထွေးသော အလုပ်များစွာကို တစ်ပြိုင်နက် လုပ်ဆောင်ရန် မဟုတ်ဘဲ စက်ပစ္စည်းများအတွင်း ထည့်သွင်း၍ ထိန်းချုပ်ရန် (Embedded Systems) အတွက် ဒီဇိုင်းထုတ်ထားခြင်း ဖြစ်သည်။

---

### သမိုင်းကြောင်း (History)
ပထမဆုံးသော Microcontroller ကို ၁၉၇၁ ခုနှစ်တွင် Intel မှ **Intel 4004** အဖြစ် စတင်မိတ်ဆက်ခဲ့သော်လည်း ၁၉၇၄ ခုနှစ်တွင် Texas Instruments မှ ထုတ်လုပ်သော **TMS 1000** သည် လက်တွေ့အသုံးချနိုင်သော ပထမဆုံး Microcontroller ဖြစ်လာခဲ့သည်။ ၁၉၉၀ ပြည့်နှစ်များနောက်ပိုင်းတွင် **Atmel (AVR)** နှင့် **Microchip (PIC)** တို့၏ ပေါ်ထွက်လာမှုကြောင့် အီလက်ထရွန်းနစ် ဝါသနာရှင်များနှင့် အင်ဂျင်နီယာများအကြား Microcontroller အသုံးပြုမှုသည် အလွန်ကျယ်ပြန့်လာခဲ့သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Microcontroller သည် အဆင့် (၃) ဆင့်ဖြင့် အလုပ်လုပ်သည် -

* **အချက်အလက်ဖတ်ခြင်း (Input):** အပြင်ဘက်ရှိ ခလုတ်များ၊ Sensor များမှ လျှပ်စစ်အချက်ပြလှိုင်းများကို Input ငုတ်များမှတစ်ဆင့် ဖတ်ယူသည်။
* **တွက်ချက်ဆုံးဖြတ်ခြင်း (Processing):** ဖတ်ယူရရှိသော ဒေတာများကို CPU အတွင်းရှိ ရေးသားထားသော Program (Code) အရ တွက်ချက်ဆုံးဖြတ်သည်။
* **တုံ့ပြန်လုပ်ဆောင်ခြင်း (Output):** ဆုံးဖြတ်ချက်အတိုင်း မော်တာများကို လည်စေခြင်း၊ LED မီးများ လင်းစေခြင်း သို့မဟုတ် Display ပေါ်တွင် စာသားပြသခြင်းတို့ကို လုပ်ဆောင်သည်။

---

### အဓိကပါဝင်သော အစိတ်အပိုင်းများ (Key Components)
* **CPU (Central Processing Unit):** ညွှန်ကြားချက်များကို အကောင်အထည်ဖော်ပေးသည့် "ဦးနှောက်"။
* **RAM (Random Access Memory):** တွက်ချက်နေစဉ်အတွင်း ဒေတာများကို ခေတ္တသိမ်းဆည်းသည့်နေရာ။
* **Flash Memory (ROM):** ရေးသားထားသော ပရိုဂရမ်များကို အမြဲတမ်း သိမ်းဆည်းထားသည့်နေရာ။
* **I/O Ports:** ပြင်ပပစ္စည်းများနှင့် ချိတ်ဆက်ရန် အဝင်/အထွက် ငုတ်များ။
* **Timers/Counters:** အချိန်ကိုက် လုပ်ဆောင်ချက်များနှင့် အကြိမ်အရေအတွက် ရေတွက်ရန်။
* **ADC (Analog to Digital Converter):** Sensor များမှလာသော Analog ဗို့အားကို ဒစ်ဂျစ်တယ်တန်ဖိုးသို့ ပြောင်းလဲပေးရန်။

---

### အသုံးများသော Microcontroller များနှင့် Board များ
1. **Arduino (Atmega328P):** သင်ယူရလွယ်ကူပြီး ဝါသနာရှင်များ အသုံးအများဆုံး။
2. **ESP32 / ESP8266:** Wi-Fi နှင့် Bluetooth ပါဝင်သဖြင့် IoT (Internet of Things) လုပ်ငန်းများတွင် သုံးသည်။
3. **STM32:** စက်မှုလုပ်ငန်းသုံးနှင့် မြန်နှုန်းမြင့် လုပ်ဆောင်ချက်များအတွက် သုံးသည်။
4. **PIC:** စက်ရုံသုံး ထိန်းချုပ်မှုစနစ်များတွင် ကြာရှည်ခံမှုအတွက် သုံးသည်။

---

### အသုံးချပုံများ (Applications)
* **အိမ်သုံးပစ္စည်းများ_** မိုက်ခရိုဝေ့ဖ်မီးဖို၊ အဝတ်လျှော်စက်နှင့် ရေခဲသေတ္တာများ၏ အလိုအလျောက် စနစ်များ။
* **မော်တော်ယာဉ်စနစ်များ_** အင်ဂျင်ထိန်းချုပ်မှု၊ လေအိတ်စနစ်နှင့် ဘရိတ်စနစ် (ABS) များ။
* **စက်ရုပ်နည်းပညာ_** စက်ရုပ်လက်တံများ ထိန်းချုပ်ခြင်းနှင့် အလိုအလျောက် လမ်းကြောင်းရှာ စနစ်များ။
* **ကျန်းမာရေးသုံးပစ္စည်းများ_** သွေးပေါင်ချိန်တိုင်းစက်နှင့် နှလုံးခုန်နှုန်းတိုင်းစက်များ။
