## Flash Memory (မြန်နှုန်းမြင့် အစုအဖွဲ့လိုက်မှတ်ဉာဏ်)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Flash Memory ဆိုသည်မှာ လျှပ်စစ်သုံး၍ ဖျက်ပစ်နိုင်ခြင်းနှင့် အသစ်ပြန်ရေးနိုင်ခြင်းတို့ ပြုလုပ်နိုင်သော **Non-volatile Storage** (ဓာတ်အားမရှိသော်လည်း ဒေတာမပျက်သော) အမျိုးအစား ဖြစ်ပါသည်။ ၎င်းသည် EEPROM ၏ အမျိုးကွဲတစ်ခု ဖြစ်သော်လည်း EEPROM ကဲ့သို့ တစ်ချက်ချင်း (Byte-by-byte) မဖျက်ဘဲ ဒေတာများကို အစုအဖွဲ့လိုက် (**Block အလိုက်**) တစ်ပြိုင်နက် "Flash" ကဲ့သို့ ဖျက်ပစ်နိုင်သောကြောင့် အဆပေါင်းများစွာ ပိုမိုမြန်ဆန်ပါသည်။

---

### သမိုင်းကြောင်း (History)
Flash Memory ကို ၁၉၈၀ ပြည့်နှစ်တွင် Toshiba ကုမ္ပဏီ၌ အလုပ်လုပ်နေသော **Dr. Fujio Masuoka** က စတင်တီထွင်ခဲ့သည်။ "Flash" ဟူသော အမည်ကို သူ၏လုပ်ဖော်ကိုင်ဖက် Mr. Shoji Ariizumi က အကြံပေးခဲ့ခြင်းဖြစ်ပြီး ဒေတာများကို ဖျက်ပစ်သည့်အရှိန်မှာ ကင်မရာမီးပွင့် (Flash) သည့်အလား မြန်ဆန်သောကြောင့် ပေးခဲ့ခြင်းဖြစ်သည်။ ၁၉၈၈ တွင် Intel က NOR Flash ကို ထုတ်လုပ်ခဲ့ပြီး ၁၉၈၉ တွင် Toshiba က NAND Flash ကို စီးပွားဖြစ် ထုတ်လုပ်ခဲ့သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Flash Memory သည် **Floating-Gate Transistor** များစွာကို စနစ်တကျ စီတန်း၍ ဒေတာများကို သိမ်းဆည်းသည် -

* **သိမ်းဆည်းမှုစနစ်-** ထရန်စစ္စတာအတွင်းရှိ Floating Gate ထဲတွင် အီလက်ထရွန်များကို ပိတ်လှောင်ထားခြင်းဖြင့် ဒေတာများကို Binary 1 သို့မဟုတ် 0 အဖြစ် သတ်မှတ်သည်။
* **Block Erase:** Flash Memory ၏ ထူးခြားချက်မှာ ဒေတာများကို အပိုင်းလိုက် (Block လိုက်) တစ်ပြိုင်နက် ဖျက်ပစ်နိုင်ခြင်း ဖြစ်သည်။ ၎င်းအတွက် **Quantum Tunneling** ဟုခေါ်သော နည်းစဉ်ကို အသုံးပြုသည်။
* **သက်တမ်း-** ဒေတာများကို ထပ်ခါတလဲလဲ ရေးသားဖျက်ပစ်ခြင်းက Chip ၏ သက်တမ်းကို တဖြည်းဖြည်း လျော့နည်းစေသည်။ ထို့ကြောင့် ခေတ်ပေါ် Storage များတွင် "**Wear Leveling**" ဟုခေါ်သော နည်းပညာကို သုံးပြီး ဒေတာများကို နေရာအနှံ့ မျှဝေရေးသားကာ သက်တမ်းရှည်အောင် လုပ်ဆောင်ကြသည်။

---

### Flash Memory အမျိုးအစားများ
1. **NAND Flash:** ပမာဏများပြားသော ဒေတာများကို သိမ်းဆည်းရန် သုံးသည်။ ရေးခြင်းနှင့် ဖျက်ခြင်း အလွန်မြန်သည်။ SSD များ၊ USB Stick များနှင့် SD ကတ်များတွင် သုံးသည်။
2. **NOR Flash:** ဒေတာဖတ်နှုန်း အလွန်မြန်ပြီး CPU က ၎င်းထဲမှ Code များကို တိုက်ရိုက်ဖတ်ပြီး အလုပ်လုပ်နိုင်သည်။ BIOS chip များနှင့် Firmware သိမ်းဆည်းရန် သုံးသည်။

---

### အသုံးချပုံများ (Applications)
* **လူသုံးကုန်ပစ္စည်းများ-** ကင်မရာမှတ်ဉာဏ်ကတ်များ၊ USB drive များနှင့် စမတ်ဖုန်းများ။
* **ကွန်ပျူတာနည်းပညာ-** Hard Disk (HDD) များအစား ပိုမိုမြန်ဆန်သော **Solid State Drive (SSD)** များအဖြစ် သုံးခြင်း။
* **Embedded စနစ်များ-** Microcontroller များ၏ Operating System နှင့် Code များကို သိမ်းဆည်းခြင်း။
* **ဆက်သွယ်ရေးစနစ်-** Router နှင့် Switch များ၏ Configuration ဖိုင်များကို သိမ်းဆည်းခြင်း။
