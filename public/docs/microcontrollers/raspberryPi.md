## Raspberry Pi (ခရက်ဒစ်ကတ် အရွယ်အစားရှိ ကွန်ပျူတာ)

### ခြုံငုံသုံးသပ်ချက် (Overview)
Raspberry Pi ဆိုသည်မှာ ယူကေနိုင်ငံရှိ Raspberry Pi Foundation မှ ထုတ်လုပ်ထားသော ခရက်ဒစ်ကတ် (Credit Card) အရွယ်အစားသာရှိသည့် ကွန်ပျူတာပြား (**Single-board Computer**) လေးများ ဖြစ်ပါသည်။ ၎င်းသည် အရွယ်အစား သေးငယ်သော်လည်း Monitor တစ်လုံး၊ Keyboard၊ Mouse တို့နှင့် ချိတ်ဆက်လိုက်ပါက သာမန် Desktop ကွန်ပျူတာတစ်လုံးကဲ့သို့ အင်တာနက်ကြည့်ခြင်း၊ စာရိုက်ခြင်းနှင့် ဂိမ်းကစားခြင်းများကို အပြည့်အဝ လုပ်ဆောင်နိုင်ပါသည်။ အထူးသဖြင့် ကျောင်းသားများနှင့် ဝါသနာရှင်များ ပရိုဂရမ်ရေးသားခြင်းကို လွယ်ကူစွာ လေ့လာနိုင်ရန် ဖန်တီးထားခြင်း ဖြစ်သည်။

---

### သမိုင်းကြောင်း (History)
ဤပရောဂျက်ကို **Eben Upton** နှင့် အဖွဲ့က ကိန်းဘရစ်ချ်တက္ကသိုလ်တွင် ၂၀၀၀ ပြည့်နှစ်အစောပိုင်းကတည်းက စတင်ခဲ့သော်လည်း ပထမဆုံး Model ကို ၂၀၁၂ ခုနှစ်၊ ဖေဖော်ဝါရီလတွင် စတင်ထုတ်လုပ်ခဲ့သည်။ မူလက ကျောင်းနေအရွယ် ကလေးငယ်များကို ကွန်ပျူတာသိပ္ပံ သင်ကြားပေးရန် ရည်ရွယ်ခဲ့သော်လည်း ဈေးနှုန်းသက်သာပြီး အသုံးဝင်လှသဖြင့် ကမ္ဘာတစ်ဝန်းရှိ အီလက်ထရွန်းနစ် ပညာရှင်များနှင့် အင်ဂျင်နီယာများအကြား အလွန်ရေပန်းစားသွားခဲ့သည်။ ယခုအခါ **Raspberry Pi 5** အထိ မျိုးဆက်များစွာ ထွက်ရှိထားပြီးဖြစ်သည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
Raspberry Pi သည် သာမန် PC များတွင် သုံးသော x86 architecture အစား ဖုန်းများတွင် သုံးလေ့ရှိသော **ARM Architecture** အပေါ် အခြေခံ၍ အလုပ်လုပ်သည် -

* **System on a Chip (SoC):** CPU၊ GPU နှင့် RAM အားလုံးကို Chip ပြားတစ်ခုတည်းပေါ်တွင် စုစည်းထားသဖြင့် ဓာတ်အားစားသုံးမှု အလွန်နည်းသည်။
* **MicroSD Card မှ အလုပ်လုပ်ခြင်း-** ၎င်းတွင် Hard drive မပါဝင်ဘဲ လိုအပ်သော Operating System (အများအားဖြင့် Linux အခြေခံ **Raspberry Pi OS**) ကို MicroSD ကတ်ထဲတွင် ထည့်သွင်း၍ အလုပ်လုပ်ရသည်။
* **GPIO ချိတ်ဆက်မှု-** သာမန်ကွန်ပျူတာများနှင့် မတူဘဲ ၎င်းတွင် **GPIO pins** များ ပါဝင်သောကြောင့် Sensor များနှင့် မော်တာများကို တိုက်ရိုက်ထိန်းချုပ်နိုင်သည်။

---

### အသုံးများသော Raspberry Pi အမျိုးအစားများ
1. **Raspberry Pi 5:** အမြင့်ဆုံးနှင့် အမြန်ဆုံး အမျိုးအစား ဖြစ်ပြီး PCIe 2.0 စနစ် ပါဝင်လာသည်။
2. **Raspberry Pi 4:** 4K monitor နှစ်လုံးအထိ ချိတ်ဆက်နိုင်ပြီး RAM 8GB အထိ ရရှိနိုင်သည်။
3. **Raspberry Pi Zero 2 W:** အလွန်သေးငယ်ပြီး ဈေးသက်သာသော အမျိုးအစား ဖြစ်သည်။
4. **Raspberry Pi Pico:** ၎င်းသည် ကွန်ပျူတာမဟုတ်ဘဲ Arduino ကဲ့သို့သော **Microcontroller** အမျိုးအစား ဖြစ်သည်။

---

### အသုံးချပုံများ (Applications)
* **Home Media Center:** TV ကို Smart TV အဖြစ် ပြောင်းလဲပြီး ရုပ်ရှင်နှင့် သီချင်းများကို ကြည့်ရှုခြင်း။
* **Retro Gaming:** ယခင်ခေတ်ဟောင်း ဂိမ်းစက်များကို ပြန်လည်အသက်သွင်းပြီး ဂိမ်းကစားခြင်း (ဥပမာ - RetroPie)။
* **Personal Server:** ကိုယ်ပိုင် Website လွှင့်ခြင်း သို့မဟုတ် ဖိုင်များကို အင်တာနက်မှတစ်ဆင့် သိမ်းဆည်းသည့် (Cloud Storage) စနစ် တည်ဆောက်ခြင်း။
* **စက်ရုပ်နည်းပညာ-** AI နှင့် Computer Vision စနစ်များကို သုံးပြီး အလိုအလျောက် သွားလာနိုင်သော စက်ရုပ်များ တည်ဆောက်ခြင်း။
