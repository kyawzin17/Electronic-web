## PIC Microcontroller (အရွယ်စုံသုံး ထိန်းချုပ်ကိရိယာ)

### ခြုံငုံသုံးသပ်ချက် (Overview)
PIC (**Peripheral Interface Controller**) ဆိုသည်မှာ Microchip Technology မှ ထုတ်လုပ်သော Microcontroller မိသားစု ဖြစ်ပါသည်။ မူလက General Instrument မှ စတင်ထုတ်လုပ်ခဲ့ခြင်း ဖြစ်ပြီး Harvard Architecture ပေါ်တွင် အခြေခံ၍ တည်ဆောက်ထားသည်။ ဈေးနှုန်းသက်သာခြင်း၊ လွယ်ကူစွာ ဝယ်ယူရရှိနိုင်ခြင်းနှင့် အစီအစဉ် (Code) ရေးသားရာတွင် အဆင်ပြေခြင်းတို့ကြောင့် ကမ္ဘာတစ်ဝန်းရှိ အီလက်ထရွန်းနစ် ဝါသနာရှင်များနှင့် အင်ဂျင်နီယာများအကြား အလွန်ရေပန်းစားသည်။ ၎င်းတွင် 8-bit မှသည် 32-bit အထိ အမျိုးအစားစုံလင်စွာ ရှိပြီး အရုပ်လေးများမှသည် ကားစက်ပိုင်းဆိုင်ရာ ထိန်းချုပ်မှုများအထိ နေရာစုံတွင် အသုံးပြုနိုင်သည်။

---

### သမိုင်းကြောင်း (History)
ပထမဆုံး PIC (PIC1650) ကို ၁၉၇၆ ခုနှစ်တွင် General Instrument ကုမ္ပဏီမှ CPU ၏ ဝန်ထုပ်ဝန်ပိုးများကို ကူညီလုပ်ဆောင်ပေးရန် စတင်ထုတ်လုပ်ခဲ့သည်။ ၁၉၈၉ ခုနှစ်တွင် Microchip Technology က ဆက်ခံခဲ့ပြီး နည်းပညာများကို ပိုမိုမြှင့်တင်ခဲ့သည်။ ၁၉၉၀ ပြည့်နှစ်များတွင် ထွက်ရှိလာသော **PIC16F84** သည် အီလက်ထရွန်းနစ်လောကတွင် အထူးထင်ရှားခဲ့ပြီး Flash memory ပါဝင်သော ပထမဆုံး ဈေးသက်သာသည့် chip ဖြစ်လာခဲ့သည်။ ၎င်းကြောင့် အသုံးပြုသူများသည် ဈေးကြီးသော စက်ကိရိယာများ မလိုဘဲ ကုဒ်များကို အကြိမ်ပေါင်းများစွာ ဖျက်လိုက်၊ ပြန်သွင်းလိုက်ဖြင့် စမ်းသပ်နိုင်လာခဲ့ကြသည်။

---

### အလုပ်လုပ်ပုံ (Working Principle)
PIC Microcontroller သည် **RISC** (Reduced Instruction Set Computer) စနစ်ကို အသုံးပြု၍ အလုပ်လုပ်သည် -

* **Harvard Architecture:** ညွှန်ကြားချက် (Instructions) သိမ်းသည့်နေရာနှင့် အချက်အလက် (Data) သိမ်းသည့်နေရာကို သီးခြားစီ ခွဲထားသောကြောင့် CPU က အလုပ်နှစ်ခုကို တစ်ပြိုင်နက် လုပ်ဆောင်နိုင်ပြီး ပိုမိုမြန်ဆန်သည်။
* **Instruction Pipelining:** ညွှန်ကြားချက်တစ်ခုကို အလုပ်လုပ်နေစဉ်မှာပင် နောက်ထပ်လုပ်ရမည့် တစ်ခုကို ကြိုတင်ယူဆောင် (**Fetch**) ထားသည့် စနစ်ဖြစ်သည်။
* **Clock Cycles:** 8-bit PIC အများစုတွင် ညွှန်ကြားချက်တစ်ခု ပြီးမြောက်ရန် (Instruction Cycle) အတွက် ပြင်ပမှပေးသော နာရီချက်လှိုင်း (Clock) ၄ ချက် လိုအပ်သည်။

---

### PIC အမျိုးအစားများ
1. **PIC10/PIC12:** အလွန်သေးငယ်သော ငုတ် ၆ ခုမှ ၈ ခုပါ chip များ ဖြစ်ပြီး ရိုးရှင်းသော အလုပ်များအတွက် သုံးသည်။
2. **PIC16 Series:** အသုံးအများဆုံး အမျိုးအစားဖြစ်ပြီး (ဥပမာ - **PIC16F877A**) အထွေထွေသုံး ပရောဂျက်များတွင် သုံးသည်။
3. **PIC18 Series:** မှတ်ဉာဏ် ပိုများပြီး တွက်ချက်မှု မြန်ဆန်စေရန်အတွက် အဆင့်မြင့် အစိတ်အပိုင်းများ ပါဝင်သည်။
4. **dsPIC:** အချက်ပြလှိုင်းများကို ဒစ်ဂျစ်တယ်နည်းဖြင့် တွက်ချက်ခြင်း (**Digital Signal Processing**) အတွက် သီးသန့်ထုတ်ထားသည်။
5. **PIC32:** အမြင့်ဆုံး စွမ်းဆောင်ရည်ရှိသော 32-bit အမျိုးအစားများ ဖြစ်သည်။

---

### အသုံးချပုံများ (Applications)
* **လူသုံးကုန်ပစ္စည်းများ-** အဝတ်လျှော်စက်များ၊ နှိုးစက်များနှင့် မိုက်ခရိုဝေ့ဖ် မီးဖိုများ။
* **မော်တော်ကားလောက-** ကားမှန်တင်/ချစနစ်၊ ဒိုင်ခွက်ပြကွက်များနှင့် အာရုံခံကိရိယာများကို ထိန်းချုပ်ခြင်း။
* **စက်မှုလုပ်ငန်း-** အပူချိန်ထိန်းကိရိယာများ၊ မော်တာအမြန်နှုန်း ထိန်းချုပ်စနစ်များနှင့် ရိုးရှင်းသော PLC များ။
* **လုံခြုံရေးစနစ်-** ဒစ်ဂျစ်တယ် တံခါးသော့များ၊ RFID ဖတ်စက်များနှင့် လှုပ်ရှားမှုအာရုံခံ ကိရိယာများ။
